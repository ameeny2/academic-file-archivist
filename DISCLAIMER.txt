DISCLAIMER & TERMS OF USE

AFA organises files using local AI. AI can make mistakes: a file may
occasionally be classified or placed somewhere you do not expect.

AFA is designed so the user always stays in control:

  1. Nothing is moved until the user reviews the proposed plan and
     explicitly clicks "Confirm & Move".
  2. Every executed session can be reversed with Undo, which restores
     all files to their original locations.
  3. Detected duplicates are never deleted directly — they are moved to
     a recoverable trash folder inside the app's own data directory.
  4. AFA never scans or modifies system folders, program installations,
     game libraries, or code repositories (built-in protection list).
